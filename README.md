# NSE Live Spike Screener

This project provides a live NSE stock spike screener.

Frontend:
GitHub Pages
https://tapankapadia007-design.github.io/Kapadia_Wealth/

Backend:
Python FastAPI server deployed on Render.

Features:
- Live NSE stock spike detection
- Price movement scanner
- Web-based dashboard
- Mobile friendly

Tech Stack:
- Python (FastAPI)
- Zerodha Kite Connect API
- WebSocket streaming
- GitHub Pages frontend
- Render backend hosting
